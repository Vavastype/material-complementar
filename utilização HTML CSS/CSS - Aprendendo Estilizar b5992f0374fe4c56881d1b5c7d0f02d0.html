Voltando com unidades de medida vamos aplicar suas utilidades…

O que são unidade de media absolutas e relativas?

As unidades de medidas absolutas seria valores fixos sobre componentes (Ex: px cm)
Valores relativos seriam unidades que sé comportam em relação ao visualização do site, sendo maior ou menor podendo lidar com responsividade do site (Ex: % vw vh ).
/*
	Unidade de medida CSS (px, em, rem, vw, vh, vmax, vmin e %)

	>>Unidadede medida absoluta:
	
	cm	centimeters
	mm	millimeters
	in	inches (1in = 96px = 2.54cm)
	px *	pixels (1px = 1/96th of 1in) <-- Vamos utilizar com frequência Unidades PX
	pt	points (1pt = 1/72 of 1in)
	pc	picas (1pc = 12 pt)
	
	>> Unidade de medida relativa a janela do usuario:
	
	em - Relativo ao tamanho da fonte do elemento (2em significa 2 vezes o tamanho da fonte atual)
	ex - Relativo à altura x da fonte atual (raramente usado)
	ch - Em relação à largura do "0" (zero)
	rem - Relativo ao tamanho da fonte do elemento raiz
	vw - Relativo a 1% da largura da janela de visualização*
	vh - Relativo a 1% da altura da janela de visualização*
	vmin - Relativo a 1% da dimensão menor da janela de visualização*
	vmax - Relativo a 1% da dimensão maior da janela de visualização*
	% Em relação ao elemento pai
*/

Vamos utilizar unidades mais comuns para criação dos projetos (EM,REM,PX, VH, VW, %, VMIN, VMAX)

Usando unidade de media vw e vh faz alteração sobre a janela de visualização do navegador
vw → alteraria a largura sobre a janela (width)
vh → alteraria a altura sobre a janela (height)

.container{
	width:100vw;
	height:100vh;
	background-color:red; /* adicionando cor de fundo sobre a pagina */
}
Valores VMIN e VMAX são unidades que responde o max ou mínimo que deseja utilizar em seu site, muito usado para fixar elementos gerais em sua pagina…
.container{
	font-size:5vmax; /*Sé comporta com a maior visualização da pagina do site - largura*/
}
.container{
	font-size:5vmin; /*Sé comporta com a menor visualização da pagina do site - altura*/
}
html{
	height:500px; /*Com esse valor da pagina, afetara o valor height em container*/
}

.container{
	width: 300px; /*Valores absolutos fixados*/
	height: 2rem; /*Valor que captura o root da pagina que seria o html*/
}

Combinador descendente

O combinador descendente - normalmente representado por um único caractere de espaço (" ") - combina dois seletores de modo que os elementos correspondidos pelo segundo seletor sejam selecionados se tiverem um elemento ancestral (pai, pai do pai, pai do pai do pai, etc.) elemento correspondente ao primeiro seletor. Os seletores que utilizam um combinador descendente são chamados de seletores descendentes.
/*
	Captura de um elemeto filho usando o container (Elemento pai)
	para captura o h1 (elemento filho)
*/
.container.h1{
	color:red;
	font-size:100px;
}
/*
	Aqui estou falando que desejo usar container mais todos os titulos
	para adicionar color pink
*/
.conatiner, h1, h2, h3, h4, h5{
	color:pink;
}

Pseudo-classes

Uma pseudo-classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado. Por exemplo, :hover pode ser usado para alterar a cor de um botão quando o usuário passar o cursor sobre ele.
button{
	color:black;
	background-color:white;
}

button:hover{
	color:white;
	background-color:black;
	font-size:25px;
}

Display - Grid

A propriedade CSS grid é uma propriedade abreviada que define todas as propriedades explícitas e implícitas da grade em uma única declaração.
Usando a grade, você especifica um eixo usando grid-template-rows ou grid-template-columns e, em seguida, especifica como o conteúdo deve ser repetido automaticamente no outro eixo usando as propriedades implícitas da grade: grid-auto-rows, grid-auto-columns, e fluxo automático de grade.
*{
    padding: 0px;
    margin: 0px;
    margin: auto;
    background-color: #f1faee;
}

.content-info{
    display: grid;/*Aplicando display grid*/
    /*colunas                1°   2°    3°    4°                            */
    grid-template-columns: 200px 200px 200px 200px;
    /*linhas              1°   2°    3°    4°                            */
    grid-template-rows: 200px 200px 200px 200px;
}
